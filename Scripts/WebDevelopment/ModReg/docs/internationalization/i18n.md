# i18n Module

A lightweight internationalization utility for React applications.

## Features
- Global locale state.
- Reactive translations via hook.
- Key interpolation.
- Simple initialization.

## Functions

### `createI18n(config)`
Initializes the i18n service.
- `config.translations`: An object where keys are locales and values are translation maps.
- `config.defaultLocale`: The initial locale to use.

### `t(key, params?)`
Translates a key.
- `key`: The translation key (e.g., 'auth.login').
- `params`: Optional object for interpolation.

### `useTranslation()`
React hook for components. Returns `{ t, locale }`.

## Example
```tsx
import { useTranslation } from './modules/internationalization/i18n';

const Welcome = () => {
  const { t } = useTranslation();
  return <p>{t('welcome', { name: 'Ziky' })}</p>;
};
```
